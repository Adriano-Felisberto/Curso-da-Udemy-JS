# Curso de JavaScript, TypeScript e Node.js

Repositório criado para armazenar códigos, exercícios e projetos desenvolvidos durante um curso completo de **JavaScript e TypeScript**, abrangendo **front-end, back-end e full stack**.

O conteúdo vai do **básico ao avançado**, utilizando tecnologias modernas e amplamente adotadas no mercado.
